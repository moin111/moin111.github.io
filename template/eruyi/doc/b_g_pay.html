[API]发起支付

URL地址：/api.php?act=pay

请求参数说明：

字段名 变量名 示例值 类型 必填 描述
应用 app 10000 GET 是 应用ID必填参数，用于获取APP配置
订单号 order 20200222174624120 POST 是 随机生成一个订单号
账号 user 123456 POST 是 可以是账号、邮箱、手机号
支付方式 way ali POST 是 ali=支付宝，wx=微信，qq=QQ钱包
商品ID gid 1 POST 是 通过获取商品列表返回
User-Agent ua 0 POST 否 用于返回支付源码类型，0=pc(电脑扫码),1=H5(手机唤起)
时间戳 t 1582266964 POST 否 如果开启了《时间差校验》就必须传入客户端时间戳
数据签名 sign 16d1b8c03072c8c7c5ecb51e157b059a POST 否 如果开启了《数据签名》就必须传入签名数据
